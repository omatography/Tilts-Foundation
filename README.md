# Tilts Foundation
 Tilts Foundation is a NGO which helps poor students to be able access online study resources in this pandemic. This is the website of the NGO.
 Developed using bootstrap4 and simple js plugins like fontawesome, waypoint. For animation used animate.css.
 No templates were used, developed from scratch. 
 To view the site :https://omatography.github.io/Tilts-Foundation/
